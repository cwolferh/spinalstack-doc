RHN
===

When deploying Spinal Stack on Red Hat, you need to configure your RHN subscription.

Configuration
-------------

================ =============== =======
Configuration    Description     Default
================ =============== =======
rhn_registration RHN Credentials -
================ =============== =======

.. code-block:: none

  ---
  rhn_registration:
    username: joe
    password: secrete
